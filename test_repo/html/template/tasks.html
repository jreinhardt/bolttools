Bases

For some classes a backend specific geometry is missing for one or more CAD packages. $basetable

Drawings

For some classes a drawing is missing that gives a visual overview over the part and clarifies the meaning of the parameters. $missingdrawings

For other classes no svg version of the drawing is available. $missingdrawingssvg

Unsupported licenses

For some classes a unsupported license name and/or url was found. Often this is the result of typos. Check with the list in the specifications to idnetify these cases. If the license is not on the list, try contact all the copyright holder of the file and ask them to relicense this file under one of the supported licenses, or remove the file. $unsupportedlicenses
